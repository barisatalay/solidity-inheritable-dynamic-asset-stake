# solidity-inheritable-dynamic-asset-stake
Solidity Inheritable Dynamic Asset Stake

Reward tokens to be distributed to the stakers must be deposited into the contract.

# Getting Started

```
npm install
cp .env.sample .env
npx hardhat compile
```
